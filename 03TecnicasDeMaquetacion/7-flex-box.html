  2. Propiedades del padre

    • Display Flex

      Esta propiedad nos permite establecer cuál sera el contenedor de los elementos que serán flexibles. Puede ser de bloque o línea.

      display: flex | inline-flex;

    • Flex Direction

      Nos permite establecer la dirección en la que se acomodaran nuestros elementos en el eje principal, de forma horizontal (row) o vertical (column).

      Es importante mencionar que el eje principal siempre dependerá del modo de escritura

      flex-direction: row | column | row-reverse | column-reverse ;

    • Flex Wrap

      Podemos indicar si todos los elementos estarán en una línea con su máximo ancho posible o establecido (nowrap), o si queremos que respete el ancho de los elementos y éstos se posicionen en varias líneas (wrap o wrap-reverse).

      flex-wrap: nowrap | wrap | wrap-reverse;

    • Flex Flow

      Es el shorthand de flex-direction y flex-wrap.

      flex-flow: < Primer valor flex-direction > < Segundo valor flex-wrap >

    • Justify Content

      Define cómo se acomodan los elementos en el eje principal

      justify-content: flex-end | flex-start | center | space-around | space-between ;

    • Align Items

      Define cómo se acomodan los elementos en el eje transversal para cuando tenemos una línea de elementos (nowrap)

      align-items: flex-start | flex-end | center | stretch | baseline;

    • Align Content

      Define cómo se acomodan los elementos en el eje transversal para cuando tenemos varias líneas de elementos (wrap o wrap-reverse)

      align-content: flex-start | flex-end | center | stretch | space-between | space-around ;

  3. Propiedades del los hijos

    • Flex Basis

      Establece el tamaño base que tendrá un elemento flexible. Si el padre es row o row-reverse se modifica el ancho del elemento, si el padre es column o column-reverse se modifica el alto del elemento.

      flex-basis: px | em | % ;

    • Flex Grow

      Establece el factor de crecimiento que tendrá un elemento en comparacion con los demás, en caso de que sobre espacio en el contenedor y su tamaño sea diminuto, funcionan cuando los elementos están en una sola línea (flex-wrap: nowrap). No acepta números negativos.

      flex-grow: 1;

    • Flex Shrink

      Establece el factor de reducción que tendrá un elemento a comparacion con los demás, en caso de que sobre espacio en el contenedor y su tamaño sea excesivo, funcionan cuando los elementos están en una sola línea de elementos (flex-wrap: nowrap). No acepta números negativos.

      flex-shrink:1;

    • Flex

      Atajo de flex-grow, flex-shrink, flex-basis

      flex:1 1 50%;

      Tiene muchas variantes 

      • flex:<'flex-grow'>					Cuando tenemos una línea (flex-wrap:nowrap)
      • flex: <'flex-basis'>					Cuando tenemos varias líneas (flex-wrap:wrap)
      • flex: <'flex-grow'> <'flex-basis'>			Cuando tenemos una línea (flex-wrap:nowrap)
      • flex: <'flex-grow'> <'flex-shrink'>			Cuando tenemos varias líneas (flex-wrap:wrap)
      • flex: <'flex-grow'> <'flex-shrink'> <'flex-basis'>	Se especifican los tres valores
      • flex: none o ( 0 0 auto )				Ninguno
      • flex: initial o ( 0 auto )				Inicial
      • flex: auto o ( 1 1 auto )				Automático
      • flex: inherit						Heredado

    • Order

      Establece el orden que tendrá un elemento en comparacion a otros. Acepta números positivos y negativos.

      order: n;

    • Align Self

      Nos permite alínear un elemento independientemente de los demás, mismos valores que align-items

      align-self: flex-start | flex-end | center | stretch | baseline ;
